pproach 1: TF-IDF + Cosine Similarity (BEST for you)
Step 1: Treat your PDFs as “your class”
Your PDFs → Class = MINE
Other PDFs → Class = NOT MINE
Step 2: Build a single “MY DOCUMENT PROFILE”

Merge all your PDFs:

my_docs = [pdf1, pdf2, pdf3, ...]
my_corpus = " ".join(my_docs)
Step 3: Vectorize
from sklearn.feature_extraction.text import TfidfVectorizer

vectorizer = TfidfVectorizer(ngram_range=(1,2))
vectorizer.fit(my_docs + [new_doc])
Step 4: Compare similarity
from sklearn.metrics.pairwise import cosine_similarity

my_vec = vectorizer.transform([my_corpus])
new_vec = vectorizer.transform([new_doc])

score = cosine_similarity(new_vec, my_vec)[0][0]
Step 5: Decision
if score > 0.3:   # tune this threshold
    return "MINE"
else:
    return "NOT MINE"
🔥 Why this works better than keywords
Captures overall writing style
Uses all words automatically
Handles bigrams naturally
No manual weighting needed
